# Optimal Transport, Old and New

This folder contains a standalone visualization-first notebook course based on the local PDF of Cedric Villani's *Optimal Transport: Old and New*.

The notebooks use the PDF only as a source map. They rebuild the concepts in original prose with small executable models, generated diagrams, coupling checks, Wasserstein computations, displacement interpolation labs, Ricci curvature visualizations, and audit scripts.

Start here:

- `00-book-index.ipynb`
- `source-map.md`
- `AGENTS.md`

Common validation commands are listed in `AGENTS.md`.
